# Iaia Brain Coach Website

## Project Description
This project is a personal website for Iaia Colella, a Brain Performance Coach for athletes. The website provides information about coaching programs, assessments, testimonials from athletes, and contact options.

## Project Structure
```
iaiabraincoach-site
├── index.html        # Main HTML file containing the structure of the webpage
├── css
│   └── style.css     # CSS file containing styles for the webpage
├── js
│   └── main.js       # JavaScript file for future interactivity and functionality
├── assets
│   └── images        # Directory containing all image files used in the website
└── README.md         # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd iaiabraincoach-site
   ```

2. **Open the Project**
   Open the `index.html` file in your preferred web browser to view the website.

3. **Modify Styles**
   To change the appearance of the website, edit the `css/style.css` file.

4. **Add JavaScript Functionality**
   For any interactive features, add your JavaScript code in the `js/main.js` file.

5. **Images**
   Place any additional images in the `assets/images` directory and update the `index.html` file accordingly.
